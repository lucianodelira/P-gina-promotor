Como se Tornar um Afiliado

📝

1️⃣ Cadastre-se na Banca Brasileira

Clique no link e faça seu cadastro para se tornar um afiliado. É rápido e gratuito!

Cadastrar-se
📸

2️⃣ Faça um print da tela inicial do app

Após concluir o cadastro, abra o app e tire um print da tela inicial, mostrando claramente seu ID de afiliado.

📲

3️⃣ Envie o print pelo WhatsApp

Envie o print para validar seu cadastro de promotor. É a forma de confirmarmos seu ID no sistema.

Enviar Print
⏳

4️⃣ Aguarde a análise

Após o envio, nossa equipe analisará seu print pelo WhatsApp. Quando a confirmação for concluída, será necessário sair e entrar novamente no app para atualizar seu link de promotor.
